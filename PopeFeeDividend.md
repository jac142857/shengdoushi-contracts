# 教皇卡手续费分红合约说明

> 通用的质押合约的方法

## 额外变量

|  变量名   | 类型  | 备注  |
|  ----  | ----  | ----  |
| GSSToken  | address | GSS代币合约 |
| PopeToken  | address | 教皇合约 |
| PledgeToken  | address | 锁仓LP合约 |


## 额外方法

- setGSSToken 设置GSS代币合约

参数

|  参数   | 类型  | 备注  |
|  ----  | ----  | ----  |
| _token  | address | token地址 |

- setPopeToken 设置教皇合约

参数

|  参数   | 类型  | 备注  |
|  ----  | ----  | ----  |
| _token  | address | token地址 |

- setPledgeToken 设置锁仓LP合约

参数

|  参数   | 类型  | 备注  |
|  ----  | ----  | ----  |
| _token  | address | token地址 |
